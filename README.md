# Solutions for [Advent of Code 2021](https://adventofcode.com/2021)

## C

Build and run/test every day

```sh
./build.sh --test
```

Build and run/test a specific day

```sh
cd ./day-01/c/
./build.sh
./main --test
```

## Rust

Test every day

```sh
cargo test
```

Run a specific day

```sh
cargo run --bin day-01
```
